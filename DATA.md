Overview
This document provides detailed information about the Traffic Light Classification Dataset used in AFP Coursework 2.

Dataset Provenance
Source Information
Provider: University of Portsmouth, School of Computing
Module: M34132 AI Foundations in Practice
Distribution Method: Canvas Learning Management System
Academic Year: 2025-2026
Purpose: Educational coursework - Binary image classification

Attribution
This dataset was provided as part of university coursework materials and is redistributed here with proper attribution for academic reproducibility purposes only.

Class Information
Class 1: Green Traffic Lights

Label: Green
Characteristics:
Green illuminated traffic lights
Various angles and distances
Different lighting conditions (day/night)
Some with partial occlusions

Class 2: Red Traffic Lights
Label: Red
Characteristics:
Red illuminated traffic lights
Various angles and distances
Different lighting conditions
Some with partial occlusions
